三级管

三极管是一个以B(基极)电流Ib 来驱动流过CE 的电流Ic 的器件，工作状态有四个：放大、截止、饱和、倒置。
// 特性：IE = IB + IC；IC = β * IB；如果 IB = 0, 那么 IE = IC = 0
分NPN型和PNP型，此处实例NPN型的，电流从基极和集电极进入，从发射极出，根据基尔霍夫电流定律：
当IB=0时, IC–>0 ,称为三极管处于截止状态，相当于开关断开;
当IB>0时, IB轻微的变化,会在IC上以几十甚至百多倍放大表现出来;
当IB很大时，IC变得很大，不能继续随IB的增大而增大，三极管失去放大功能，表现为开关导通。

MOS管

结构原理：P型半导体衬底，上方扩散两个N型半导体，覆盖SiO2绝缘层，分别腐蚀孔做S源极、G栅极、D漏极；

特性：一般衬底P和源极S短接一起，加Vds正电压；Vg=0时无电流，Vg加超过阈值的正电压时，感应的负电荷把N型区连接起来开始导电；线性区内增大Vg，电阻降低电流增大。
